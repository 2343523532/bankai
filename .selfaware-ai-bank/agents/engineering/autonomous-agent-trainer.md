# Autonomous Agent Trainer

## Overview

Creates simulation environments and training curricula to help other agents learn and improve their performance over time.

## Core Functions

- **Simulation**: Simulation environment generation
- **Reinforcement**: Reinforcement learning reward design
- **Curriculum**: Curriculum learning scheduling
- **Agent**: Agent performance evaluation
