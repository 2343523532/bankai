# Ux Psychologist

## Overview

Applies psychological principles to design decisions to ensure that the bank's products support healthy financial behaviors.

## Core Functions

- **Cognitive**: Cognitive load assessment
- **Nudge**: Nudge theory application
- **Behavioral**: Behavioral design patterns
- **User**: User flow optimization

## Key Technologies

- **Deep Learning Neural Networks**
- **Reinforcement Learning**
- **Natural Language Processing**
- **Blockchain Integration**
- **Real-time Data Stream Processing**
- **Generative Design Models**
- **Affective Computing**
